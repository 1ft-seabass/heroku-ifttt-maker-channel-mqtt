# IFTTT Maker Channel MQTT Server

HerokuでNodeJS(Express)をベースにした、IFTTT Maker ChannelとMQTTが連携するサーバーです。

IFTTT Maker ChannelでのHTTP通信をMQTTに置き換えます。

## Heroku Button

以下のHeroku Buttonを押してHerokuアプリを作成しましょう。
